**> 无编号公式**

- `\[ 公式 \]` 或 `$$公式$$`

• 核心依赖：需要 `graphicx` 宏包（部分环境自带）。

  

• 常用代码模板：

  

```

  

\begin{figure}[htbp]

  

    \centering

  

    \includegraphics[图片大小]{图片路径}

  

    \caption{图片标题、说明}

  

    \label{图片标签}

  

\end{figure}

  

```

  

• 参数设置说明：

  

  • [图片大小]：使用 `height` 和 `width` 规定，单位可用 cm 或 in。如果只规定其中一个，图片将保持高宽比插入，一般只需规定宽度。

  

  • [图片路径]：指图片相对于 tex 文件的路径。

  

• 双栏显示特殊处理：部分双栏显示的模板，图片如果需要跨栏显示，把 `\begin{figure} ... \end{figure}` 替换为 `\begin{figure*} ... \end{figure*}` 即可。

  

• PS：尽量插入矢量图。


**更新后的笔记内容概览：**

- **核心依赖**：`graphicx` 宏包。
    
- **代码模板**：`\begin{figure}[htbp] ...`
    
- **参数设置**：图片大小 (`width`/`height`)、路径。
    
- **双栏处理**：使用 `figure*`。
    
- **`htbp` 参数详解**：
    
    - **定义**：控制浮动体位置的选项集。
        
    - **选项含义**：`h` (当前位置), `t` (顶部), `b` (底部), `p` (单独一页)。
        
    - **组合使用**：如 `ht`（首选顶部，次选当前）。默认值为 `tbp`。
        
    - **示例**：`[htbp]` 综合尝试所有位置。
