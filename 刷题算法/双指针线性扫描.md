# 844. 比较含退格的字符串

## 一、题目描述

给定两个字符串 `s` 和 `t`。

规则：

- `#` 表示 **退格（删除前一个字符）**
    
- 如果文本为空再退格，文本仍然为空
    

判断：

```text
两个字符串经过编辑器处理后是否相等
```

返回：

```text
true / false
```

---

## 二、示例

### 示例1

输入

```
s = "ab#c"
t = "ad#c"
```

处理过程

```
ab#c -> ac
ad#c -> ac
```

输出

```
true
```

---

### 示例2

输入

```
s = "ab##"
t = "c#d#"
```

处理

```
ab## -> ""
c#d# -> ""
```

输出

```
true
```

---

### 示例3

输入

```
s = "a#c"
t = "b"
```

处理

```
a#c -> c
b -> b
```

输出

```
false
```

---

## 三、解题思路

本题核心是 **模拟退格操作**。

常见两种解法：

```
1 栈 / StringBuilder（容易理解）
2 双指针（最优解）
```

---

## 四、解法一：栈模拟

思想：

```
遇到字符 → 加入
遇到 # → 删除最后一个字符
```

实现可以用：

```
StringBuilder
```

### Java代码

```java
class Solution {
    public boolean backspaceCompare(String s, String t) {
        return build(s).equals(build(t));
    }

    public String build(String str) {
        StringBuilder sb = new StringBuilder();

        for (char c : str.toCharArray()) {
            if (c != '#') {
                sb.append(c);
            } else if (sb.length() > 0) {
                sb.deleteCharAt(sb.length() - 1);
            }
        }

        return sb.toString();
    }
}
```

复杂度

```
时间 O(n)
空间 O(n)
```

---

## 五、解法二：双指针（推荐）

关键观察：

```
# 只影响前面的字符
```

所以可以：

```
从字符串末尾开始遍历
```

思路：

1. 定义两个指针
    

```
i 指向 s 末尾
j 指向 t 末尾
```

2. 遇到 `#`
    

```
记录需要跳过的字符数量
```

3. 找到真正有效字符再比较。
    

---

### Java代码

```java
class Solution {
    public boolean backspaceCompare(String s, String t) {

        int i = s.length() - 1;
        int j = t.length() - 1;

        int skipS = 0;
        int skipT = 0;

        while (i >= 0 || j >= 0) {

            while (i >= 0) {
                if (s.charAt(i) == '#') {
                    skipS++;
                    i--;
                } else if (skipS > 0) {
                    skipS--;
                    i--;
                } else {
                    break;
                }
            }

            while (j >= 0) {
                if (t.charAt(j) == '#') {
                    skipT++;
                    j--;
                } else if (skipT > 0) {
                    skipT--;
                    j--;
                } else {
                    break;
                }
            }

            if (i >= 0 && j >= 0) {
                if (s.charAt(i) != t.charAt(j)) {
                    return false;
                }
            } else {
                if (i >= 0 || j >= 0) {
                    return false;
                }
            }

            i--;
            j--;
        }

        return true;
    }
}
```

复杂度

```
时间 O(n)
空间 O(1)
```

## 六、核心技巧总结

本题关键：

```
从后往前遍历字符串
```

原因：

```
退格只影响前面的字符
```

所以：

```
逆序处理最简单
```
